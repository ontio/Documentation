### 手动创建

手动创建安装文件

### 依赖工具

* Node
* Yarn (https://yarnpkg.com/lang/en/docs/install/)

#### 安装 yarn

```
npm install --global yarn
```

#### 下载
```
git clone 'https://github.com/backslash47/ontology-plugin-wallet.git'
```


#### Build

````
yarn build
````
