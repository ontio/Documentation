Cyano 是一个Chrome插件钱包.不仅实现了钱包功能，还实现了DAPI协议，例如SmartX部署和调用合约时通过Cyano与链交互。

Cyano特点:

* 通过助记词创建钱包
* 加密私钥
* 通过Keystore创建钱包
* 支持Ledger
* 支持Trezor
* 查余额
* ONG 和 ONT转账
* 提取ONG
* 切换网络 (Test/Main/Private) ，支持 TLS
* 支持ONT ID 
* 支持 dAPI 
* NEO/ONT 地址和Ledger地址支持
